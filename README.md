## 🎯 Sobre o Projeto

O site foi projetado para ser responsivo, rápido e altamente otimizado para a conversão de orçamentos através do WhatsApp. Ele proporciona uma excelente experiência de usuário (UX) tanto em dispositivos móveis quanto em computadores, refletindo o profissionalismo e a agilidade da empresa.

> **Status do Projeto:** Concluído ✅

## � Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Estilização utilizando Flexbox, CSS Grid, Variáveis (Custom Properties) e animações em keyframes.
- **JavaScript (Vanilla):** Interatividade para o menu mobile responsivo, animações de revelação de conteúdo durante o scroll da página (utilizando `IntersectionObserver`) e formatação/validação do formulário com redirecionamento para a API do WhatsApp.

## ⚙️ Funcionalidades

- 📱 **Design 100% Responsivo:** Adaptável perfeitamente a smartphones, tablets e desktops.
- ✨ **Animações "On-Scroll":** Elementos surgem de forma suave conforme o usuário desce a página, tornando a navegação fluida.
- 💬 **Formulário Integrado ao WhatsApp:** Captura os dados essenciais do cliente (nome, telefone, e-mail e pedido) e inicia o contato de forma automatizada.
- 🔗 **Otimização para Compartilhamento (SEO / Open Graph):** Preparado com metatags (OG) para gerar miniaturas atrativas ao compartilhar o link nas redes sociais.

## 🖥️ Como rodar o projeto localmente

O projeto foi construído utilizando tecnologias web nativas, portanto, não necessita de nenhuma instalação complexa ou servidor de build.

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/lojas-amel.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd lojas-amel
   ```
3. Dê um duplo clique no arquivo `index.html` ou abra-o em seu navegador de preferência.
